# PITB Security routes

Route files here are published by `security:install` for your **mode only** (`api`, `web`, or `hybrid`).

- **API** → `security-api.php`, `security-api-auth.php`
- **Web** → `auth.php`, `security.php`, `security-admin.php`
- **Hybrid** → all of the above

See `README.md` in this folder after install (copied from the mode-specific template).

```bash
php artisan security:routes
```
